# Reporting

Follow offsec's template: 

https://help.offsec.com/hc/en-us/articles/360046787731-PEN-200-Reporting-Requirements

## MAKE SURE!

You need to include in your screenshots on how and why you got there, run these commands on the box and take a snapshot, otherwise, you will be dinged for not having it:

Linux:
```
ifconfig && id && cat /root/proof.txt
```
Windows:
All root flags are here:
```
cd C:\Users\Administrator\Desktop
```
Then:
```
ipconfig && whoami && type proof.txt
```

Constantly be taking notes as you go, get the snapshots of the flag, and you will do just fine.
